Servicio Web

Para comenzar este proyecto está divido en seis (6) partes y son:

  1. Barra de Navegación Superior
  2. Sección de Video
  3. Sección de Beneficios
  4. Sección de Planes
  5. Segmento de contacto
  6. Pie de pagina

En la barra de navegación destacamos las distintas partes a las cuales queremos navegar rápidamente, para que el usuario no tenga que hacer scroll todo el tiempo. Entre estas partes estaría el logo de la empresa, Features que son los beneficios del servicio de esta empresa, Services para destacar los distintos planes y por ultimo Contact Us para guiarnos a la sección de donde colocamos nuestro correo.

Uno de los principales problemas que se presenta es que esta parte desaparecía una vez descendíamos, sin embargo, se puede solucionar fácilmente si en nuestro archivo.css le añadimos a este elemento la propiedad position: y le asignamos el valor fixed. De esta forma la barra de navegación nos seguirá en todo momento.

Respecto a la sección de video debemos usar la funcionalidad de compartir de Youtube, la cual nos genera un código HTML dentro de etiquetas <iframe>, el cual vamos añadir a nuestra página y ajustaremos con estilos CSS. En cuanto a la sección de beneficios debemos establecer la posición tanto para los imágenes como el texto, una de las formas que se puede lograr es usando la propiedad con su valor display: grid, para volver la grilla este contenedor y luego indicamos las columnas que queremos usar con grid-template-columns.

Ahora bien, con la sección de planes queremos establecer qué servicio resaltar al colocar en el medio, de esta forma al usuario ver las otras opciones decidirá que la mejor es la que está entre todas. Por esta razón, es que se usan una cantidad de tarjetas impar es decir se pueden usar 3, 5 o 7, lo más común son 3. Para lograr este efecto debemos establecer contenedores para cada servicio los cuales vamos a estilizar usando el formato estándar de tarjetas (Si no conoces como crearlo puede consultar la forma básica de en esta publicación de W3Schools).

En relación al segmento de contacto vamos a utilizar etiquetas <form> donde utilizaremos el atributo action=”” para indicar la dirección donde queremos que se envíen los datos y el atributo action=”POST” para que se pasen de forma oculta. Recuerda, que el objetivo principal de esta sección es obtener un medio de contacto con el usuario para brindarle un excelente servicio, asi que solo necesitaremos dos <input> uno para el nombre y el otro para correo o número telefonico.

Como última parte tendremos el pie de página, donde dispondremos enlaces de interés para el usuario y otros medios de contacto para obtener más información.
